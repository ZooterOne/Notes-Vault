## Operators

`site:<domain>`
`intext:<string>`
`filetype:<extension>`
`intitle:<string>`
`before:<yyyy-mm-dd>`
`after:<yyyy-mm-dd>`
`location:<string>`
`inurl:<string>`

___

## Examples

```
site:pastebin.com intext:"admin password"
```
```
intitle:"webcamxp 5"
```
```
filetype:env "DB_PASSWORD"
```
```
intitle:"Nessus Scan Report" "This file was generated by Nessus"
```
