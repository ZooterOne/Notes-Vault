### Resize image
```
magick <source> -resize <percent>% <destination>
```
resize an image.
___
### Virtual box
```
sudo mount -t vboxsf -o uid=1000,gid=1000 <share name> <folder>
```
Enable a shared folder from a virtual machine.
___
### File properties
```
pdfinfo <file>
```
display pdf properties.
```
exiftool <file>
```
display exif properties.
