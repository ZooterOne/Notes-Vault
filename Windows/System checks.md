### Energy
```
powercfg /energy
```
report energy problems.
```
powercfg /batteryreport
```
generate a battery report.
___
### File system
```
chkdsk /f
```
check file system and fix errors.
```
chkdsk /r
```
check physical sector issues and recover information.
```
sfc /scannow
```
check system files and repair problems.
