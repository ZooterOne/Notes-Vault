## Generate hash

```
md5sum <file> [> <file>]
```

calculate the MD5 hash of a file.

```
sha1sum <file> [> <file>]
```

calculate the SHA1 hash of a file.

```
sha256sum <file> [> <file>]
```

calculate the SHA256 hash of a file.

___

## Check hash

```
md5sum -c <md5 file>
```

check MD5 hash of a file.

```
sha256sum -c <sha256 file>
```

check SHA256 hash of a file.
